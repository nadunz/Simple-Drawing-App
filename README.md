# Simple-Drawing-App
This is a simple application to draw somthing on a canvas in multiple colors. It has cappable to draw by dragging , clicking mouse and pressing button.

## Testing
- Compiling
      `javac DrawingApp.java` 
  
- Running
      `java DrawingApp`
